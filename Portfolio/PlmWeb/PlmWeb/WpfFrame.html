Read your favorite articles, XAML WPF Application

The used can read, write and do various searching and sorting operations of his favorite articles. There are also books and author lists which also could be operated.

MVVM Pattern

The application is based on the MVVM design pattern.

Classes structure

The main three objects – Article, Book and Author are separated into different classes. The main class for storing articles is ArticleStoreViewModel. The other classes AuthorViewModel and BookViewModel inherits the ArticleStoreViewModel class.

Class diagram

The operation logic is based on INotifyPropertyChanged and ICommand interfaces. The class structure is standard for the MVVM pattern.

Operations

The operations and the behavior are designed in the XAML section via bindings and ICommand implemented classes.

Data storage

The data is stored, read and operated via System.Xml.Linq and separate class DataPersister.

Optimize Your Health, XAML WPF Application

The application is based on five mind maps and some healthy living good practices. The user can navigate between these five mind maps, each with several groups and items, reading their content and based on existing or a custom topic the user can create an event (alarm) to be notified one before and at the event raising.

Data storage

The data is loaded via BubbleTreeLoader class and xml file storage operating class.

Notification icons design

Notification objects inherits INotifyPropertyChanged and are executed with event handler at given time. The UI design is with animation in the XAML part.

Reminder / Timer classes

The reminder logic is implemented with event handlers and an additional library Quartz.

Searches

The user can search in the database with two kind of methods - > fast recursive search, and linear loop search.

Sorting by properties

The sorting is implemented by ListSortDirection class by columns in both directions.
