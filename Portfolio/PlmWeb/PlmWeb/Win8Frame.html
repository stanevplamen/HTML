Optimize Your Health, Windows 8 Store Application

The application is based on five mind maps and some healthy living good practices. The user can navigate between five main set of articles, each with several groups and items. Notification and searching classes are also implemented. The data is generated from xml file.

MVVM Pattern

The application is based on the MVVM design pattern.

Classes diagram and structure

The main parent class BindableBase implements INotifyPropertyChanged and it is inherited by the common view model SampleDataCommon class, which is parent for Item, Group and GroupStore view models.

Operations

The operations and the behavior are designed in the XAML section via bindings and ICommand implemented classes.

Data storage

The data is stored asynchronously, read and operated via System.Xml.Linq and separate class DataPersister in local folder.

Searching and Events models

Searching and Events view models are also implemented. Again with inheritance if the BaseViewModel, INotifyPropertyChanged ,Observable Collections of items and Operations via ICommand interface.
